# Wyre Innovations Website

A clean, modern, mobile-first static website built with VitePress for Wyre Innovations - a leading manufacturer of clean energy solutions.

## 🌟 Features

- **Mobile-First Design**: Responsive across all devices with Apple-inspired minimalist aesthetic
- **Clean Energy Focus**: Showcasing sine wave inverters, solar inverters, MPPT controllers, and Li-ion BMS
- **SEO Optimized**: Comprehensive meta tags, structured data, and search engine optimization
- **Dark/Light Mode**: Toggleable theme with smooth transitions
- **Fast Performance**: Built with VitePress for optimal static site generation
- **Automated Deployment**: GitHub Actions workflow for seamless GitHub Pages deployment

## 🛠️ Technology Stack

- **Framework**: [VitePress](https://vitepress.dev/) - Vue.js powered static site generator
- **Styling**: Custom CSS with Apple-inspired design system
- **Fonts**: Inter with Lovelo fallback for branding
- **Deployment**: GitHub Pages with automated CI/CD
- **SEO**: Structured data, Open Graph, Twitter Cards

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/wyre-innovations/wyre-innovations.github.io.git
cd wyre-innovations.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## 🎨 Design System

The website follows Apple's design principles:

- **Typography**: Inter font family with careful spacing and hierarchy
- **Colors**: Adaptive light/dark color palette
- **Components**: Rounded corners, subtle shadows, smooth transitions
- **Layout**: Clean, spacious layouts with generous white space
- **Mobile-First**: Responsive design optimized for all screen sizes
